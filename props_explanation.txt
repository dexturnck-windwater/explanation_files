
1. What are Props?

Props (short for properties) are used in React to pass data from one component to another. Props go from parent to child. Props help make components reusable. Props are read-only, which means they cannot be changed inside the child component.

2. Simple Example

Parent Component:

<User name="Johnson" />

Child Component:

function User(props) {
  return <h1>Hello, {props.name}</h1>;
}

Output:
Hello, Johnson

3. Why Use Props?

Props allow you to reuse the same component with different data. They help keep your code simple and organized. They also help separate logic and display.

4. Multiple Props Example
<User name="Johnson" age={15} />
function User(props) {
  return (
    <div>
      <p>Name: {props.name}</p>
      <p>Age: {props.age}</p>
    </div>
  );
}
5. Props with Destructuring (Optional Advanced)
function User({ name, age }) {
  return <p>{name} is {age} years old</p>;
}
